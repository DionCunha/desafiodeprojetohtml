A tag h1, h2, ... ,h6

As tags h1, h2, h3 etc são as tags responsáveis por hierarquizar a tipografia dos títulos dos textos. Seria o equivalente ao título, subtítulo, subsubtítulo dos "estilos" na formatação de um documento no word. Por exemplo, a escrita em html : <ul> <li><h1>h1 Título</h1></li> <li><h2>h2 Subtítulo</h2></li> <li><h3>h3 Subsubtítulo</h3></li> <li><h4>h4 Subsubsubtítulo</h4></li> <li><h5>h5 Subsubsubsubtítulo</h5></li> <li><h6>h6 Subsubsubsubsubtítulo</h6> </li> </ul> gera o seguinte texto:

  • h1 Título

  • h2 Subtítulo

  • h3 Subsubtítulo

  • h4 Subsubsubtítulo

  • h5 Subsubsubsubtítulo
  • h6 Subsubsubsubsubtítulo

A tag "p"

A tag "p" é usada pelo desenvolvedor do html para escrever os parágrafos do texto do site. Ou seja, para escrever os parágraafos, o desenvolvedor do html deve escrever as palavras que formarão o texto do parágrafo entre os comandos <p></p>.

(Voltar)

A tag "mark"

A tag "mark" é usada para fazer uma marcação de texto com uma cor. Por exemplo, ao usar o comando <mark>palavra com marcador de texto</mark> o seguinte resultado é exibido na página do site: palavra com marcador de texto

(Voltar)

A tag "strong"

A tag "strong" é usada para se escrever uma palavra em negrito. O comando <strong>palavras em negrito</strong> gera o seguinte resultado: palavras em negrito

(Voltar)

A tag "small"

A tag "small" é usada quando o desenvolvedor do site quer usar uma fonte de letra menor num intervalo definido de um texto. Para escrever uma palavra com uma fonte de letra menor, devemos escrever a palavra dentro do seguinte código: <small></small>

(Voltar)

A tag "ol"

A tag "ol" (ordered list) é usada para se fazer uma lista ordenada. Os itens da lista, nessa tag, ficam numerados de em ordem crescente. Como exemplo veja os comandos que devemos usar para visualizar o seguinte conteúdo:

  1. Região Norte
  2. Região Nordeste
  3. Região Sudeste
  4. Região Sul
  5. Região Centro-Oeste
<ol> <li>Região Norte</li> <li>Região Nordeste</li> <li>Região Sudeste</li> <li>Região Sul</li> <li>Região Centro-Oeste</li> </ol>

(Voltar)

A tag "ul"

Já a tag "ul" (unordered list) cria uma lista não ordenada. Os itens da lista não ordenada não são numerados. Tomando como referência o comando da tag "ol", podemos repetir os comandos, tomando o cuidado apenas de trocar o por u. Assim, teremos:

  • Região Norte
  • Região Nordeste
  • Região Sudeste
  • Região Sul
  • Região Centro-Oeste
<ul> <li>Região Norte</li> <li>Região Nordeste</li> <li>Região Sudeste</li> <li>Região Sul</li> <li>Região Centro-Oeste</li> </ul>

(Voltar)

A tag "li"

É com o uso dessa tag, "li" que os itens de uma lista, seja ela ordenada ou não ordenada, são exibidos. A tag "li" deve ser escrita dentro do ambiente <ol></ol> ou <ul></ul>. Por exemplo, a lista com os sete estados da região norte do Brasil pode ser assim executada em html: <ul> <li>Acre</li> <li>Amapá</li> <li>Amazonas</li> <li><strong>Pará</strong></li> <li>Rondônia</li> <li>Roraima</li> <li>Tocantins</li> </ul> para que vejamos na página do site o seguinte conteúdo:

  • Acre
  • Amapá
  • Amazonas
  • Pará
  • Rondônia
  • Roraima
  • Tocantins

A tag "a"

A tag "a", ou também conhecida como tag de âncora, daí o uso da letra a, é usada para se fazer ancoragem entre elementos que pertencem ao site em desenvolvimento, tanto pode ser elemento interno ao site quanto externo. Para conseguirmos efetivar o uso dessa tag, devemos usar o atributo href e uma frase ou palavra dentro da tag <a></a>.

(Voltar)

A tag "i"

A tag "i" é usada para fazer uma palavra ficar em formato itálico, ou seja, o comando <i>itálico</i> gera o seguinte resultado: itálico.

(Voltar)

A tag "u"

A tag "u" é uma tag de formatação de texto. Ela é usada para sublinhar um palavra ou frase. É usada assim: <u>esta frase está sublinhada</u> para produzir o seguinte resultado: esta frase está sublinhada

(Voltar)

A tag "sub"

A tag "sub", em inglês subscript (subscrito), é usada para fazer, por exemplo, uma letra ou número ficar abaixo da linha base da palavra no texto. Acompanhe o exemplo: H2O. Repare que o algarismo 2 está abaixo da linha base das letras H e O. Os comandos usados para o exemplo da fórmula química da água foi obtido a partir de: H<sub>2</sub>O.

(Voltar)

A tag "sup"

A tag "sup", em inglês superscript (sobrescrito), é usada para "fazer" expoentes alfa-numéricos. Veja os exemplos: X2+X-1=0, May2024, 11th. Os códigos que geraram os exemplos da equação do 2º grau e da data, em inglês, foram: X<sup>2</sup>+X-1=0, May2024, 11<sup>th</sup>.

(Voltar)

A tag "blockquote"

A tag "blockquote" é usada para fazer uma citação. Por exemplo: Friedrich Nietzsche foi um visionário filósofo alemão que desafiou as convenções de sua época. Sua obra, marcada por conceitos como "além do bem e do mal", "vontade de potência" e "eterno retorno" questiona os fundamentos da moralidade e da existência humana. Muitas vezes incompreendido em vida, hoje o pensamento de Nietzsche tem grande influência no pensamento moderno, destacando-se pela ousadia em questionar verdades que até então eram inquestionáveis.

"Às vezes, as pessoas não querem ouvir a verdade porque não querem que suas ilusões sejam destruídas."
